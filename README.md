# Silly snake game 

on rxjs 5 (https://github.com/ReactiveX/rxjs) and ramda (http://ramdajs.com/docs/)

## Installation

* Install node, npm
* `git clone`
* `cd snake`
* `npm i`
* `npm run serve` to compile and run
* `npm run build` just to compile src and generate bundle.js file 
inside the dist dir
